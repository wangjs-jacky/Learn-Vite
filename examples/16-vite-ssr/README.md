## 前言

本项目基于 `vite` 搭建 `SSR` 框架



## 使用教程

1. 使用 `npm run dev` 启动项目， `ssr` 框架处于开发阶段，服务端读取的 `src` 下的入口文件。
2. 使用 `npm run build` 先构建 `client/ssr` 代码，使用 `npm run preview` 启动项目，`ssr` 框架处于生产阶段，服务端读取的 `dist` 打包后的产物。



启动成功后，就可以看到 `vite` 由脚手架创建的初始界面：
